# Reggie is Regex NextJS site

Here is a simple integration showing how to load a soul into a provider, dispatch events to it, and receive events from it. It incorporates [Reggie The Regex](../README.md) as its soul.

## Getting Started

1. Deploy the soul to your Soul Engine accoutn (see the [higher level readme](../README.md)).
2. Edit [.env.local](./.env.local) to hold your Soul Engine Organization.
3. `npm run dev`
   

Check out the [components](./src/components/) directory to see the details of the integration.
